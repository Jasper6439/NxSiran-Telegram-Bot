# v1.6.2.2 文档化 + 代码清理 设计文档

## 目标

为项目一级代码目录生成 README.md 概要表格，同时清理低于 v1.3 的过时版本注释。

### README.md 格式

每个 README 包含：
1. 目录用途一句话描述
2. 文件概要表格（文件名 | 功能描述 | 主要导出）
3. 依赖关系说明（可选，仅复杂的目录）

### 代码清理

| 操作 | 文件 | 详情 |
|---|---|---|
| 删除低版本注释 | `game_api/media_routes.py` | 文件头标注 v0.3，需更新或删除版本标记 |
| 检查 Skill 标记注释 | `packages/commands/` 下多个文件 | `[Skill: xxx]` 标记属于开发阶段遗留，需清理 |
