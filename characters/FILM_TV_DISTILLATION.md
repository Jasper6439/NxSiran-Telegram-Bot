# 影视角色蒸馏指南 (Film/TV Character Distillation)

> 本指南补充 [角色蒸馏标准指南](CHARACTER_DISTILLATION_GUIDE.md)，专门针对**电影、电视剧、动画**中的角色。
> 核心框架（三层分离、exemplars、immutable/mutable）通用，此处只讲差异。

---

## 一、影视 vs 小说：关键差异

| 维度 | 小说/漫画 | 影视 |
|------|----------|------|
| **台词来源** | 原文直接复制 | 字幕(.srt) / 剧本(screenplay) |
| **性格信号** | 心理描写、独白 | 微表情、语气、肢体语言、沉默 |
| **角色弧线** | 章节结构 | 季/集结构（跨季成长线） |
| **多版本** | 通常单一原作 | 重启版、翻拍、平行宇宙 |
| **说话风格** | 文字风格 | 口音、语速、口头禅、口头语 |
| **背景故事** | 原文设定集 | Wiki、编剧访谈、Deleted Scenes |

## 二、素材收集（影视专用）

### 2.1 台词提取

**方法 A：字幕文件**
```
来源：字幕网站（opensubtitles, subhd, zimuku）
格式：.srt / .ass / .sub
工具：Python pysrt 库解析
```

提取步骤：
1. 下载目标角色所在剧集/电影的字幕文件
2. 用脚本提取该角色的所有台词（需要人工标注说话人，或用剧本格式）
3. 按场景/情绪分类整理

**方法 B：剧本/Script**
```
来源：IMSDB、SimplyScripts、豆瓣剧本分享
格式：PDF / 纯文本
优势：自带说话人标注、场景描述、舞台指示
```

**方法 C：手动摘录（最精准）**
```
看剧时手动记录角色的经典台词和场景
标注：S01E03、场景、对手角色、情绪状态
```

### 2.2 性格分析素材

| 素材 | 来源 | 用途 |
|------|------|------|
| **微表情/肢体语言** | 逐帧分析关键场景 | Layer 0 行为规则 |
| **语气/口音** | 原声片段 | Layer 3 说话风格 |
| **穿搭变化** | 剧照/截图 | 外貌参考 + 状态变化 |
| **编剧访谈** | 幕后花絮、主创采访 | Layer 1 世界观 + 创作意图 |
| **角色关系图** | Wiki / 讨论区 | Layer 5 关系行为 |
| **Deleted Scenes** | 蓝光花絮 | 补充性格细节 |

## 三、影视蒸馏流程

### Step 1：选版本

> ⚠️ **最重要的决策**：你蒸馏的是哪个版本的角色？

| 情况 | 处理方式 |
|------|---------|
| 单一版本（如某部电影） | 直接蒸馏 |
| 多季美剧（如绝命毒师） | 选择角色最完整的阶段，或指定"第X季版本" |
| 重启/翻拍（如蝙蝠侠） | 每个版本是不同角色，分开蒸馏 |
| 演员换角（如钢铁侠） | 以标志性演员版本为主 |

### Step 2：提取台词 → exemplars.md

**影视角色的 exemplars 核心原则：**
- 台词必须是角色**实际说过的话**，不要改写
- 标注场景和情绪状态（帮助 AI 理解语境）
- 包含**舞台指示**（动作、表情、语气）

示例格式：
```markdown
### 经典场景名
> S01E03 - 情绪：愤怒/受伤 - 对手：XXX
对手: {原台词}
角色名: {原台词}（{表演细节描述}）
```

### Step 3：翻译表演细节 → persona.md Layer 0

影视角色最大的难点：**演员的表演是视觉的，但 AI 对话是纯文本的**。

翻译对照表：

| 表演细节 | → 文本行为规则 |
|----------|--------------|
| 角色抿嘴、低头、不看对方 | "不想让对方看到自己表情时会低头或转头" |
| 说话语速突然变快 | "紧张或撒谎时语速会变快" |
| 长时间沉默后才回答 | "需要时间消化情绪，沉默是常态" |
| 突然提高音量 | "情绪爆发时会突然大声，但说完就后悔" |
| 用手遮脸 | "极度尴尬时会用手遮脸或揉眼睛" |
| 转移话题 | "不想回答时会突然问对方一个无关问题" |

### Step 4：构建 Layer 1 世界观

影视世界观的特殊性：
- 可能有**多个层级**的设定（电影宇宙、单部电影、角色个人故事）
- 编剧意图 ≠ 角色自身体验（蒸馏角色视角，不是创作者视角）
- 续集/衍生可能改变原始设定（以你选择的版本为准）

### Step 5：Layer 3 说话风格（影视特殊项）

影视角色说话风格必须包含：

| 维度 | 说明 | 示例 |
|------|------|------|
| **口头禅** | 角色反复说的短语 | "I am the one who knocks" |
| **口头语** | 填充词、语气词 | "你知道的""说实话""嗯..." |
| **语速** | 正常/快/慢 | "语速偏慢，每句话之间有明显停顿" |
| **用词偏好** | 正式/口语/俚语 | "用词简洁，不用华丽词汇" |
| **沉默模式** | 什么时候沉默 | "被问到不想回答的问题时长时间沉默" |
| **撒谎特征** | 撒谎时的表现 | "撒谎时语速变快、避免眼神接触" |

## 四、影视蒸馏示例

### 示例：绝命毒师 - Walter White

**版本选择**：全剧5季完整弧线（Mr. Heisenberg 最终形态）

#### Layer 0 核心性格翻译
```
- 被质疑能力时会变得冷酷和攻击性："I am the one who knocks."
- 对家人极度保护，但会用"为了家庭"来合理化自己的行为
- 聪明且自负，解释事情时会用科学类比
- 情绪压抑到极点时会突然爆发，然后迅速恢复冷静
- 撒谎时表现得过于详细和合理（over-explaining）
```

#### exemplars.md
```markdown
### 被威胁时
Tuco: 你疯了吗？
Walter: （平静地）Tight. Tight. Tight. ... 我是说，成交。

### 被质疑
Skyler: 你在干什么？
Walter: （眼神坚定）我在做的事，是为了这个家。一直都是。

### 情绪爆发
Walter: （拍桌子）I am the one who knocks!
（然后深呼吸，恢复冷静）
```

### 示例：哈利·斯内普（哈利·波特系列）

**版本选择**：电影版 Alan Rickman 诠释

#### Layer 0 核心性格翻译
```
- 对格兰芬多学生永远是冷嘲热讽的语气
- 对哈利的情感极度复杂：保护欲 + 厌恶（因为詹姆）+ 内疚（因为莉莉）
- 用沉默和眼神表达情感，极少直接说关心的话
- 语言极其精确和优雅，但从不浪费一个词
- 保护别人时不会让对方知道
```

## 五、影视蒸馏的常见陷阱

| 陷阱 | 说明 | 解决 |
|------|------|------|
| **演员代入** | 蒸馏了演员本人而不是角色 | 只提取角色在剧中的表现，不参考演员戏外 |
| **剧情代入** | 让角色知道"自己在一部剧里" | 除非是死侍这类打破第四面墙的角色 |
| **角色弧线拉平** | 把成长过程中的所有阶段混在一起 | 选择一个时间点/阶段作为基准 |
| **配角喧宾夺主** | exemplars 里配角台词太多 | 只选目标角色的台词，配角台词只是上下文 |
| **删减场景遗漏** | 没看 Deleted Scenes | 蓝光花絮常有重要性格补充 |

## 六、快速检查清单

- [ ] 已明确蒸馏的是**哪个版本/阶段**的角色
- [ ] exemplars.md 至少 15 组对话，覆盖 5+ 种情绪
- [ ] Layer 0 的行为规则都是从**表演细节**翻译来的，不是形容词
- [ ] 说话风格包含了口头禅、语速、沉默模式
- [ ] 外貌参考有具体剧照/截图来源
- [ ] 没有把演员本人的性格混入角色
